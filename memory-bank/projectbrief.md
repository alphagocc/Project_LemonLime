# Project LemonLime - 项目基础文档

## 项目概述
Project LemonLime 是一个基于 Qt 的竞赛编程评测系统，专为编程竞赛设计，支持多题目、多测试用例的自动化评测。

## 核心需求
- 支持多种编程语言（C/C++/Java/Python等）
- 自动化编译和运行测试用例
- 实时评测结果显示
- 支持比赛模式和时间限制
- 提供详细的评测报告和统计信息
- 跨平台支持（Windows/Linux/macOS）

## 项目目标
- 为编程竞赛提供稳定可靠的评测环境
- 简化比赛组织流程
- 提供直观的用户界面
- 确保评测的准确性和公平性
- 支持大规模并发评测

## 技术栈
- 框架：Qt 5.x/6.x
- 构建系统：CMake
- 日志系统：spdlog
- 单实例支持：SingleApplication
- 国际化支持：Qt Linguist

## 项目结构
- src/ - 源代码目录
- assets/ - 资源文件（图标、图片等）
- translations/ - 国际化翻译文件
- manual/ - 用户手册
- cmake/ - CMake构建配置
- 3rdparty/ - 第三方依赖库

## 开发规范
- 遵循 Qt 编码规范
- 使用 CMake 进行构建管理
- 支持多语言国际化
- 保持跨平台兼容性
- 使用 Git 进行版本控制

## 项目状态
当前为活跃开发状态，持续进行功能增强和bug修复。
